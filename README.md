# ai-final-project

Goal: Code different AI algorithms for Connect 4 (Reinforcement Learning, Minimax, AlphaBeta Pruning, Monte Carlo Tree Search, etc.) and compare/contrast them.

Steps:
1. Connect 4 interface
2. Create different AI strategies
3. Make it playable
4. Add a graphical interface
5. Write up project report
6. Slides

AI Algorithms:
- Minimax w/ AlphaBeta Pruning
- Monte Carlo Tree Search
- Reinforcement Learning
- Simple Heuristic Search

## Installation Instructions

### 1. Create virtual environment
```sh
python3 -m venv .venv # or python -m venv .venv
```
### 2. Activate .venv
```sh
source .venv/bin/activate
```
### 3. Install Requirements
```sh
pip install -r requirements.txt
```
### 4. Run `demo.py`:
```sh
python demo.py
```
